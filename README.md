# Predictive-Sales-Model-for-Retail-Inventory-Optimization-
Analyse retail sales data to identify declining SKU trends and forecast top-selling items for optimal inventory management.
Jupyter Notebook File Name: project_sales_forecasting.ipynb 
Description: This notebook comprises all data processing stages, including data cleaning, exploratory data analysis (EDA), SKU analysis, and sales predictions. Each code cell in the notebook is accompanied by comments explaining the purpose of the code.
Analysis Report File Name: Sales_Data_Analysis_and_Forecasting_Report.pdf Description: This document contains a detailed report of all the processes undertaken and discusses the findings from the analysis. The report is structured to provide insights into the data handling and forecasting results systematically.
